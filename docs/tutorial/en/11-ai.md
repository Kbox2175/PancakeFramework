## Overview

`pancake-ai` provides a unified LLM client, memory management, and RAG (Retrieval-Augmented Generation).

Supported providers: OpenAI, DeepSeek, Gemini, Ollama, Zhipu GLM, Moonshot, Qwen, vLLM.

## Enable

```xml
<dependency>
    <groupId>io.pancake</groupId>
    <artifactId>ai</artifactId>
</dependency>
```

## Configuration

```yaml
# src/resource/yaml/ai.yaml
ai:
  default_model: deepseek
  providers:
    deepseek:
      type: openai
      base_url: https://api.deepseek.com
      api_key: ${DEEPSEEK_API_KEY}
      model: deepseek-chat
      max_tokens: 4096
      temperature: 0.7
    openai:
      type: openai
      base_url: https://api.openai.com/v1
      api_key: ${OPENAI_API_KEY}
      model: gpt-4o
    gemini:
      type: google
      api_key: ${GEMINI_API_KEY}
      model: gemini-2.0-flash
    ollama:
      type: ollama
      base_url: http://localhost:11434
      model: llama3
  memory:
    short_term:
      backend: memory      # memory | redis | mybatis
      max_messages: 20
      ttl: 86400
    long_term:
      backend: memory
      ttl: 0               # 0 = never expires
  rag:
    backend: pgvector       # pgvector | redis | mongodb
    chunk_size: 500
    top_k: 5
```

## Chat

```python
# Basic chat
response = await chat_model.chat([{"role": "user", "content": "Hello"}])

# Specify model
response = await chat_model.chat(
    [{"role": "user", "content": "Explain quantum mechanics"}],
    model="gemini"
)

# Streaming
async for chunk in chat_model.chat_stream([{"role": "user", "content": "Write a poem"}]):
    print(chunk, end="")
```

## Short-Term Memory

```python
# Add messages
short_term_memory.add("user", "My name is Alice")
short_term_memory.add("assistant", "Hello Alice!")
short_term_memory.add("user", "I'm 25 years old")

# Get message list (auto-manages context window)
messages = short_term_memory.get_messages()

# Clear memory
short_term_memory.clear()
```

## Long-Term Memory

```python
# Store
await long_term_memory.remember("user_name", "Alice")
await long_term_memory.remember("user_age", 25)

# Retrieve
name = await long_term_memory.recall("user_name")  # "Alice"

# Delete
await long_term_memory.forget("user_age")
```

## RAG (Retrieval-Augmented Generation)

```python
# Add documents
await rag.add_document("Pancake is a decorator-driven Python framework")
await rag.add_document("Pancake's core is the Dough IoC container")

# Ask questions
answer = await rag.ask("What is Pancake?")
# Auto-retrieves relevant docs, combines with LLM to generate answer
```

## Custom Provider

```python
from pancake_ai import BaseProvider, register_provider

class MyProvider(BaseProvider):
    async def chat(self, messages, **kwargs):
        return "Hello from MyProvider"

    async def chat_stream(self, messages, **kwargs):
        yield "Hello "
        yield "from MyProvider"

register_provider("my_provider", MyProvider)
```
