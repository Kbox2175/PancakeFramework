## Recommended Learning Path

```
1. Quick Start       → Get started with the framework
2. Core Concepts     → Understand Dough, lifecycle, scopes
3. IoC Container     → Master dependency injection and Bean management
4. Configuration     → Learn XML/YAML configuration
5. Plugin System     → Understand plugin architecture
6. Embed             → Enable zero import (highly recommended)
7. MyBatis           → Database ORM
8. Web               → Web server development
9-17. Other Plugins  → Learn as needed
```
