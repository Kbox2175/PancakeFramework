## 推荐学习路径

```
1. 快速开始      → 了解框架基本用法
2. 核心概念      → 理解 Dough、生命周期、作用域
3. IoC 容器      → 掌握依赖注入和 Bean 管理
4. 配置系统      → 学习 XML/YAML 配置
5. 插件系统      → 理解插件架构
6. Embed         → 启用零 import（强烈推荐）
7. MyBatis       → 数据库 ORM
8. Web           → Web 服务器开发
9-17. 其他插件   → 按需学习
```
